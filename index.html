Annotald by Jana Beck, Aaron Ecay, and Anton Karl Ingason

Modifying IcePaHC's annotation tool, Annotald, for use outside of Icelandic.

Annotald is a software tool for annotating phrase-structure treebanks in the Penn Treebank style. Information on Annotald's requirements and use is available here.

Dependencies

  • Python 2
  • CherryPy 3
  • NLTK
  • Google Chrome
